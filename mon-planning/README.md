# Planning Hebdomadaire Interactif

Application web pour organiser sa semaine visuellement.

## Fonctionnalités

- Planning Lundi-Samedi avec cartes colorées et drag & drop
- Panneau Notes & Priorités (haute / moyenne / basse)
- To-Do List avec catégories et glisser-déposer vers le planning
- Export PDF du planning
- Sauvegarde automatique (localStorage)
- Restauration des jours supprimés

## Lancer le projet

```bash
npm install
npm run dev
```

## Stack

React 19 · Vite 7 · html2pdf.js
